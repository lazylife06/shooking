# 개발 환경 설정 및 규칙
1. react 설치
   - 작업용 폴더 생성
   - 작업폴더에 shift + 우클릭해서 여기서 powershell 열기
   - npm create vite@latest 입력
2. 스토리북 설치
   - 터미널에 npx storybook@latest init 입력
3. 커밋 메세지 규칙
  - 변경된 부분의 함수명과 간략한 내용 적기
