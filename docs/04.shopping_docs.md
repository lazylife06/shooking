# 쇼핑몰 연동

## 프로젝트 개요
- 쇼핑몰의 각 페이지의 데이터가 서로 연동되고 자연스럽게 넘어갈 수 있게 할 수 있도록 구현
- 참여자: 김지호

## 개발 환경
React, JavaScript

## 설치 및 실행 방법

```
npm install
npm start
```

## 프로젝트 구조

## 주요 함수와 기능
- Root(): 라우팅하는 모든 페이지의 시작
- ErrorPage(): 에러 처리 페이지

## 테스트 방법

```
npm test
```
